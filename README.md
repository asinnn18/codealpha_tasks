# Power BI Analytics Portfolio – CodeAlpha Internship

## Overview
This repository contains three Power BI dashboard projects completed as part of the CodeAlpha Internship Program.

The projects focus on solving real-world business problems using data visualization, analytics, and insight-driven decision-making.

Dashboards included in this repository:
1. Financial Health Dashboard  
2. Human Resources Analytics Dashboard  
3. Educational Performance and Resource Allocation Dashboard  

---

## Task 1: Financial Health Dashboard

### Objective
To analyze an organization’s financial performance and provide real-time insights for improved financial planning and strategic decision-making.

### Key Features
- Income statement analysis  
- Balance sheet overview  
- Cash flow monitoring  
- Revenue versus expense trends  
- Profitability analysis  
- KPI cards (Revenue, Profit, Growth Percentage)  
- Forecasting using Power BI analytics tools  

### Insights Generated
- Identified revenue growth patterns  
- Analyzed expense-heavy categories  
- Evaluated liquidity and financial stability  
- Forecasted future financial trends  

---

## Task 2: Human Resources Analytics Dashboard

### Objective
To analyze workforce data and optimize HR decision-making through performance tracking, attrition analysis, and predictive insights.

### Key Features
- Employee headcount overview  
- Attrition rate analysis  
- Department-wise performance analysis  
- Salary distribution insights  
- Hiring and exit trends  
- Workforce planning predictions  

### Insights Generated
- Identified departments with high turnover  
- Analyzed the relationship between salary and attrition  
- Evaluated employee performance trends  
- Observed workforce growth patterns  

---

## Task 4: Educational Performance and Resource Allocation Dashboard

### Objective
To evaluate student academic performance and optimize institutional resource allocation using data-driven insights.

### Key Features
- Subject-wise academic performance analysis  
- Attendance tracking  
- Pass percentage monitoring  
- Teacher and resource allocation overview  
- Budget utilization insights  
- Department and class comparisons  

### Insights Generated
- Identified subjects with low performance rates  
- Analyzed attendance impact on academic outcomes  
- Highlighted resource allocation gaps  
- Evaluated budget efficiency  

---

## Tools and Technologies Used
- Power BI Desktop  
- DAX (Data Analysis Expressions)  
- Power Query  
- Data Modeling  
- Interactive visualizations and slicers
